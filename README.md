## 《凡人修仙》经营

为了保障消费者权益，保证消费服务有效性，

有主动经营“修仙”者，可提交相关审核资料。

1.服务器有效期至少半年，拥有服务器持有证明。

2.游戏群体可达 30 人以上，且为该群拥有者。

3.管理服务至少 12h 以上，且拥有管理成员。

4.消费服务体系相关制度及其说明文档。

审核通过后,每月收益额按版权费 5%-15%交付。

经营者前期可获得工作室相关流量扶持、

拥有问题咨询优先处理、更新要求优先处理等相关服务。

> 未经审核私自提供消费服务者,黑名单处理。
