# Yunzai Next Xiuxian

## 部署

- 安装项目

```sh
yarn add yz-xiuxian -W
```

- yunzai.config.js

```ts
import { defineConfig } from 'yunzai'
export default defineConfig({
  // 在配置中，增加模块
  applications: ['yz-xiuxian']
})
```

发送`/个人信息`开启存档并展示游玩指令
