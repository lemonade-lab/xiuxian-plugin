## 安装教程      

>Yunzai-Bot/目录下执行      
```
git clone  https://gitee.com/ningmengchongshui/Xiuxian-Plugin-Box.git ./plugins/Xiuxian-Plugin-Box/   
```
>查看具体功能  #修仙帮助     
>查看扩展功能  #修仙扩展     
>查看版本说明  #修仙版本   
```
npm i js-yaml -w
npm i yamljs -w
```  
>部分出现缺少module依赖      
>请自行根据云崽安装情况       
>把npm更改为pnpm或cnpm等
