# YunzaiJS Xiuxian

## 部署

- 安装项目

```sh
yarn add yz-xiuxian jsxp -W
```

- 加载模块

`yunzai.config.json`

```ts
{
 "applications":['yz-xiuxian']
}
```

- 开始

发送`/个人信息`开启存档并展示游玩指令
