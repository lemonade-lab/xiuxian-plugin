# 云崽修仙插件

这是一款为云崽设计的修仙插件

使用typescript进行编写。在使用前，需要进行编译后才能使用。

- 源码

```sh
git clone  --depth=1 https://gitee.com/ningmengchongshui/xiuxian-plugin.git ./plugins/xiuxian-plugin
```

- 进入目录

```
cd ./plugins/xiuxian-plugin
```

- 安装依赖

```
pnpm install 
```

- 执行打包

```
npm run build
```
